== Stack

Running on podman:

- Prometheus
- Grafana

Gather metrics with blackbox-exporter

podman run -d -p 9090:9090 -v prom-data:/etc/prometheus quay.io/prometheus/prometheus
podman run -d --name=grafana -p 3000:3000 docker.io/grafana/grafana-enterprise
